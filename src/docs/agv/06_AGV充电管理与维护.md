# AGV 充电管理与维护指南

## 概述

本文档涵盖 AGV 电池充电管理的标准操作流程、充电桩维护以及电池保养知识。

## 1. 电池系统概述

### 1.1 电池类型

本项目使用的 AGV 电池为 **磷酸铁锂电池（LiFePO4）**：

| 参数 | 规格 |
|------|------|
| 额定电压 | 48V |
| 额定容量 | 30Ah / 50Ah |
| 充电电压 | 54.4V - 58.4V |
| 放电截止电压 | 40V |
| 循环寿命 | > 2000 次 |
| 工作温度 | -20°C - 55°C |

### 1.2 电池管理系统（BMS）

BMS 提供以下保护功能：

```
□ 过充保护（充电电压 > 58.8V 停止）
□ 过放保护（放电电压 < 40V 停止）
□ 过流保护（放电电流 > 100A 限流）
□ 短路保护（检测到短路立即断开）
□ 温度保护（> 60°C 停止充放电）
□均衡功能（电芯电压均衡）
```

## 2. 充电操作流程

### 2.1 自动充电流程

当 AGV 检测到电池电量低于 20% 时，自动执行回充：

```
1. AGV 检测到低电量
2. 向调度系统申请回充
3. 调度系统分配充电桩
4. AGV 导航至充电桩
5. 精确定位后对接充电接口
6. 开始充电（指示灯：紫色闪烁）
7. 充满后进入保持模式（指示灯：绿色常亮）
8. 等待调度系统下发新任务
```

### 2.2 手动充电流程

**操作步骤：**

```
1. 将 AGV 行驶至指定充电区域
2. 确认 AGV 完全停止
3. 按下急停按钮
4. 打开充电接口盖板
5. 插入充电枪（听到"咔哒"声）
6. 启动充电桩
7. 确认充电状态正常
8. 填写充电记录表
```

### 2.3 充电状态判断

| 指示灯状态 | 含义 |
|------------|------|
| 🔴 红色闪烁 | 充电故障 |
| 🟡 黄色闪烁 | 预充电中 |
| 🔵 蓝色闪烁 | 正常充电中 |
| 🟢 绿色常亮 | 充电完成 |
| 🟢 绿色闪烁 | 保持充电 |

## 3. 充电参数设置

### 3.1 充电策略

| 模式 | 电量范围 | 策略说明 |
|------|----------|----------|
| 正常充电 | 20% - 100% | 电量低于 20% 回充，充至 100% |
| 快速充电 | 20% - 80% | 仅充至 80%，缩短充电时间 |
| 定时充电 | 可设置时间 | 利用低谷电价时段 |
| 维护充电 | 定期执行 | 长期存放时每月一次 |

### 3.2 充电时间估算

```
公式：充电时间 = 电池容量 ÷ 充电电流

示例（30Ah 电池，10A 充电电流）：
充电时间 = 30Ah ÷ 10A = 3 小时（0-100%）
快速充电 = 30Ah × 0.6 ÷ 10A = 1.8 小时（20%-80%）
```

### 3.3 充电桩参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 充电电流 | 10A / 15A / 20A | 可根据电池规格调整 |
| 浮充电压 | 54.4V | 充满后维持电压 |
| 均衡电流 | 0.5A | 电芯均衡充电 |
| 温度补偿 | -3mV/°C | 环境温度补偿 |

## 4. 充电桩维护

### 4.1 日点检项目

```
□ 充电桩外观清洁
□ 指示灯状态正常
□ 充电接口无烧蚀
□ 散热风扇运转正常
□ 接地线连接可靠
□ 显示屏幕无异常
```

### 4.2 周保养项目

1. **清洁保养**
   ```
   - 使用干燥软布擦拭外壳
   - 使用压缩空气清理散热孔
   - 清理充电接口氧化物
   - 检查防水密封性
   ```

2. **电气检查**
   ```
   - 测量输入电压：220V ± 10%
   - 测量输出电压：54.4V ± 1%
   - 检查接线端子紧固
   - 测试急停功能
   ```

### 4.3 月检修项目

| 项目 | 方法 | 标准 |
|------|------|------|
| 绝缘电阻 | 500V 摇表测量 | > 10MΩ |
| 接地电阻 | 接地电阻测试仪 | < 4Ω |
| 充电效率 | 功率计测量 | > 90% |
| 冷却系统 | 目视 + 温度测试 | 温度正常 |

### 4.4 充电桩故障处理

**故障：充电桩不启动**
```
排查步骤：
1. 检查电源开关是否闭合
2. 检查输入电压是否正常
3. 检查急停按钮是否复位
4. 检查充电枪是否到位
5. 检查 AGV 充电接口
```

**故障：充电电流不稳定**
```
排查步骤：
1. 检查充电线连接是否松动
2. 测量输入电压是否波动
3. 检查环境温度是否过高
4. 排除电池 BMS 保护
```

## 5. 电池保养规范

### 5.1 新电池初始化

新电池或长时间未使用的电池：

```
步骤：
1. 首次充电必须充满（充至 100%）
2. 进行一次完整的充放电循环
3. 放电至 10-20% 再充满
4. 记录初始容量数据
5. 前 10 次充电不要深度放电
```

### 5.2 日常使用保养

```
✓ 尽量避免电池电量低于 10%
✓ 不使用时保持 50% - 80% 电量存放
✓ 充电环境温度：10°C - 40°C
✓ 避免高温下充电（> 45°C）
✓ 每月进行一次均衡充电
```

### 5.3 长期存放保养

| 存放时间 | 存放电量 | 保养要求 |
|----------|----------|----------|
| 1 周内 | 任意 | 无特殊要求 |
| 1-3 个月 | 50% - 80% | 每月检查电量 |
| 3 个月以上 | 50% - 80% | 每半月充电一次 |
| 重新启用 | 先充满 | 进行一次完整循环 |

### 5.4 电池更换标准

出现以下情况应更换电池：

```
□ 实际容量 < 标称容量 70%
□ 充电时间显著延长（> 150% 正常时间）
□ 充电后电量下降速度明显加快
□ 电池外观膨胀或变形
□ BMS 通讯故障无法修复
□ 严重过放导致电芯损坏
```

## 6. 安全注意事项

### 6.1 充电安全守则

```
⚠ 充电区域必须通风良好
⚠ 禁止在充电区域使用明火
⚠ 禁止在充电时拆解电池
⚠ 充电时禁止启动 AGV
⚠ 充电接口有水禁止充电
⚠ 发现异常立即停止充电
```

### 6.2 应急处理

**电池起火处理：**
```
⚠ 保持安全距离（至少 5 米）
⚠ 使用 CO2 灭火器（禁用水）
⚠ 疏散人员至上风口
⚠ 报警并通知专业消防队
⚠ 切勿自行扑救
```

**电解液泄漏处理：**
```
⚠ 穿戴防护手套和护目镜
⚠ 使用中和剂（碳酸氢钠）处理
⚠ 大量清水冲洗
⚠ 废液按规定收集处理
```
