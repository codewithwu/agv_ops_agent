# AGV 定期检修计划

## 概述

本文档规定 AGV 设备的定期检修计划和时间节点，确保设备处于良好运行状态。

## 1. 检修周期总览

| 检修类型 | 周期 | 预计时长 | 负责人员 |
|----------|------|----------|----------|
| 日点检 | 每日 | 15 分钟 | 操作员 |
| 周保养 | 每周 | 1 小时 | 运维人员 |
| 月检修 | 每月 | 2-3 小时 | 维护技师 |
| 季度保养 | 每季度 | 半天 | 维护技师+厂家 |
| 年度大修 | 每年 | 1-2 天 | 厂家工程师 |
| 非计划检修 | 按需 | 不确定 | 视情况 |

## 2. 日点检流程

### 2.1 点检时间

```
常规安排：
- 早班开机前：6:30 - 6:45
- 晚班交接前：18:30 - 18:45

特殊情况：
- 设备长时间停用后启用
- 故障修复后运行
- 环境异常后（暴雨、高温等）
```

### 2.2 点检内容

```
□ 电池电量检查（> 50% 方可运行）
□ 急停按钮测试（每台 AGV）
□ 外观目视检查
□ 运行区域障碍物检查
□ 通讯状态确认
□ 警示灯/声音报警确认
□ 轮胎外观检查
□ 传感器镜头清洁度检查
```

## 3. 周保养流程

### 3.1 保养时间

每周五下午或停产日进行，预计 1 小时/台

### 3.2 保养项目

**机械部分：**
```
□ 驱动轮磨损检查（测量花纹深度）
□ 转向轮灵活性检查
□ 轮轴润滑情况检查
□ 制动器间隙调整
□ 紧固件扭矩检查
□ 减震器检查
```

**电气部分：**
```
□ 电池端电压测量
□ 充电接口清洁和检查
□ 接线端子紧固检查
□ 电缆磨损检查
□ 接地电阻测量
□ 各端口连接检查
```

**传感器部分：**
```
□ 激光雷达镜头清洁
□ 超声波传感器清洁
□ 碰撞传感器灵敏度测试
□ 编码器信号测试
□ 陀螺仪零点检查
```

## 4. 月检修流程

### 4.1 检修时间

每月最后一个周末或停产日进行，预计 2-3 小时/台

### 4.2 检修项目

**性能测试：**
```
□ 最高速度测试（空载）
□ 制动距离测试
□ 定位精度测试
□ 障碍检测距离测试
□ 电池容量测试（放电测试）
□ 充电效率测试
```

**深度检查：**
```
□ 电机绝缘电阻测试
□ 电机三相电流平衡测试
□ 控制板各输出测试
□ 安全回路测试
□ 通讯信号强度测试
□ 软件版本检查
```

**参数校准：**
```
□ 里程计校准
□ 陀螺仪零偏校准
□ 激光雷达角度校准
□ 障碍检测阈值校准
□ 停车精度校准
```

## 5. 季度保养

### 5.1 保养时间

每季度末进行，需要半天时间

### 5.2 保养内容

```
□ 月检修全部项目
□ 电机深度保养
□ 变速箱检查（如果有）
□ 电池深度保养
□ 安全系统深度测试
□ 调度系统联动测试
□ 历史故障数据分析
□ 备件库存检查
```

### 5.3 厂家支持项目

| 项目 | 内容 | 是否需要厂家 |
|------|------|--------------|
| 电机深度检测 | 振动分析、温度曲线 | 是 |
| 控制板诊断 | 深度故障扫描 | 是 |
| 导航算法优化 | 参数调优 | 是 |
| 安全认证 | 符合性检查 | 是 |

## 6. 年度大修

### 6.1 大修时间

每年一次或累计运行超过 2000 小时

### 6.2 大修项目清单

```
一、拆解检查
□ 驱动电机拆解检查
□ 转向机构拆解检查
□ 变速箱检查（如有）
□ 车轮轴承更换
□ 密封件更换

二、电气系统
□ 控制板全面检测
□ 电机绝缘老化评估
□ 电缆老化检查
□ 电池组容量评估
□ BMS 功能测试

三、安全系统
□ 安全回路全面测试
□ 急停系统测试
□ 碰撞传感器校准

□ 系统升级评估
```

## 7. 非计划检修

### 7.1 触发条件

```
□ 故障修复性检修
□ 碰撞后检修
□ 异常告警排查
□ 运行参数异常
□ 安全测试不通过
□ 长期闲置后启用
```

### 7.2 处理流程

```
1. 故障报告/异常发现
2. 初步诊断
3. 制定检修方案
4. 安排检修时间
5. 实施检修
6. 测试验证
7. 恢复运行
8. 记录归档
```

## 9. 关键指标追踪

### 9.1 设备效率指标

| 指标 | 计算公式 | 目标值 |
|------|----------|--------|
| 可用率 | (运行时间 / 总时间) × 100% | > 95% |
| 故障间隔 | 两次故障间的运行时间 | > 500h |
| 故障修复时间 | 故障发生到恢复的时间 | < 2h |
| 计划停机率 | 计划停机时间 / 总时间 | < 3% |

### 9.2 维护成本指标

| 指标 | 计算公式 | 参考值 |
|------|----------|--------|
| 单位运行成本 | 维护总费用 / 运行小时 | < 10元/h |
| 备件消耗率 | 备件费用 / 运行小时 | < 5元/h |
| 非计划维修率 | 非计划维修次数 / 总维修次数 | < 20% |
