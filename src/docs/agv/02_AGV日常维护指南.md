# AGV 日常维护指南

## 概述

本文档规定 AGV 日常维护的标准流程和检查项目，确保设备稳定运行。

## 1. 定期维护周期

| 维护类型 | 周期 | 负责人员 |
|----------|------|----------|
| 日点检 | 每次使用前 | 操作员 |
| 周保养 | 每周一次 | 运维人员 |
| 月检修 | 每月一次 | 维护技师 |
| 季深度保养 | 每季度一次 | 厂家支持 |

## 2. 日点检项目

### 2.1 外观检查

```
检查项目：
□ 车身外壳无裂纹、无变形
□ 表面清洁，无残留物料
□ 警示灯正常工作
□ 显示屏无异常
```

### 2.2 传感器检查

```
□ 激光传感器镜头清洁
□ 接近传感器无遮挡
□ 地面传感器灵敏度正常
□ 碰撞传感器响应正常
```

### 2.3 轮组检查

```
□ 驱动轮磨损度 < 5mm
□ 转向轮转动灵活
□ 轮轴无异响
□ 轮组螺栓紧固
```

### 2.4 电气系统检查

```
□ 电池电量显示准确
□ 充电接口无烧蚀
□ 电源线无破损
□ 急停按钮响应灵敏
```

## 3. 周保养项目

### 3.1 清洁工作

- 使用干燥软布擦拭车身
- 清理驱动轮和转向轮缠杂物
- 清洁传感器镜头（用专用布）
- 清理充电桩接触点氧化物

### 3.2 润滑保养

| 部位 | 润滑剂 | 方法 |
|------|--------|------|
| 轮轴轴承 | 锂基润滑脂 | 每月一次 |
| 转向机构 | 润滑油 | 每周检查 |
| 传动链条 | 链条油 | 每周一次 |

### 3.3 电气检查

- 测量电池端电压：正常范围 48-52V
- 检查接线端子：无松动、无氧化
- 测试接地电阻：< 4Ω
- 检查保险丝：规格正确，无熔断

## 4. 月检修项目

### 4.1 机械系统

```
检修项目：
□ 车轮轴承间隙调整
□ 传动系统皮带张力
□ 制动器磨损检查
□ 减震器性能测试
□ 紧固件全面检查
```

### 4.2 电子系统

```
检修项目：
□ 控制板各端口检查
□ 驱动板输出测试
□ 传感器精度校准
□ 通讯模块信号强度
□ 安全回路测试
```

### 4.3 软件系统

- 检查固件版本是否为最新
- 备份当前配置参数
- 校准里程计
- 测试各传感器阈值

## 5. 常见故障与处理

### 5.1 故障代码对照表

| 代码 | 含义 | 处理方法 |
|------|------|----------|
| E001 | 左驱动电机过载 | 检查电机温度 |
| E002 | 右驱动电机过载 | 检查电机温度 |
| E003 | 激光传感器异常 | 清洁或更换 |
| E004 | 电池电量低 | 立即充电 |
| E005 | 碰撞传感器触发 | 检查障碍物 |
| E006 | 通讯中断 | 检查网络 |
| E007 | 地图匹配失败 | 重新定位 |
| E008 | 路径规划失败 | 检查地图 |

### 5.2 简单故障自排除

**故障：AGV 行驶偏离轨迹**
```
排查步骤：
1. 检查车轮是否打滑
2. 校准陀螺仪
3. 重新扫描地面标记
4. 校验地图数据
```

**故障：障碍物检测误报**
```
排查步骤：
1. 清洁激光传感器
2. 检查反射板位置
3. 调整检测阈值
4. 排除地面反光干扰
```

## 6. 维护记录管理

### 6.1 记录要求

每次维护必须填写《AGV 维护记录表》，内容包括：
- 设备编号
- 维护日期和时间
- 维护人员签名
- 检查项目结果
- 更换配件记录
- 异常情况说明

### 6.2 记录保存

- 纸质记录保存期限：2 年
- 电子记录永久保存
- 故障记录单独归档
