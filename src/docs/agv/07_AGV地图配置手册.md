# AGV 地图配置手册

## 概述

本文档介绍 AGV 地图的创建、配置和管理，是实现 AGV 自主导航的基础。

## 1. 地图系统架构

### 1.1 地图类型

| 地图类型 | 说明 | 适用场景 |
|----------|------|----------|
| 栅格地图 | 网格化环境表示 | 激光导航 |
| 拓扑地图 | 站点和路径图 | 路径规划 |
| 特征地图 | 关键点特征 | 视觉定位 |

### 1.2 本系统地图构成

```
地图数据 = 栅格地图 + 拓扑地图 + 站点配置 + 特殊区域
```

- **栅格地图**：记录环境障碍物分布（分辨率 0.05m）
- **拓扑地图**：站点之间的连接关系
- **站点配置**：各站点的位置和属性
- **特殊区域**：限速区、禁行区、充电区

## 2. 地图创建流程

### 2.1 环境要求

进行地图构建前需确认：

```
□ 地面材质与设计一致
□ 无临时障碍物
□ 光线条件正常
□ 反射板已安装（如使用反射式激光）
□ 参考物位置已标记
```

### 2.2 建图步骤

**第一步：初始化建图**

```
1. 将 AGV 移至起点位置
2. 在调度系统中选择"地图构建"模式
3. 选择"新建地图"
4. 输入地图名称和分辨率
5. 开始手动控制 AGV 移动
```

**第二步：扫描环境**

```
手动控制 AGV 沿以下路径移动：
1. 沿边界行走一圈
2. 沿主要通道行走
3. 覆盖所有工作区域
4. 经过所有站点位置
5. 保持匀速慢速行驶（0.3-0.5 m/s）
```

**第三步：优化地图**

```
系统自动处理：
1. 点云拼接
2. 噪声过滤
3. 障碍物膨胀
4. 生成栅格地图
5. 提取特征点
```

**第四步：保存地图**

```
1. 预览生成的地图
2. 检查是否有缺失或变形
3. 标注关键位置
4. 保存地图文件
5. 上传至调度系统
```

### 2.3 建图参数说明

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 地图分辨率 | 0.05m | 越小精度越高，计算越大 |
| 点云采样间隔 | 0.02m | 影响地图精细度 |
| 障碍物膨胀半径 | 0.3m | 安全距离 |
| 特征点提取距离 | 1.0m | 定位特征间隔 |

## 3. 站点配置

### 3.1 站点类型

| 类型 | 图标 | 功能 |
|------|------|------|
| 起点 | 🏠 | 任务起始点 |
| 终点 | 🎯 | 任务目的地 |
| 中转站 | 🔄 | 路径经过点 |
| 充电站 | ⚡ | 充电对接点 |
| 等待区 | ⏸ | 临时停车点 |
| 禁止区 | 🚫 | 禁止进入 |

### 3.2 添加站点

```
操作步骤：
1. 在地图上点击目标位置
2. 选择"添加站点"
3. 输入站点信息：
   - 站点编号（唯一，如 A01）
   - 站点名称（如"原料仓库"）
   - 站点类型
   - 站点坐标（自动获取或手动输入）
4. 设置站点属性
5. 保存站点
```

### 3.3 站点属性

| 属性 | 必填 | 说明 |
|------|------|------|
| 站点编号 | ✓ | 唯一标识 |
| 站点名称 | ✓ | 显示名称 |
| 站点类型 | ✓ | 功能分类 |
| X 坐标 | ✓ | 地图 X 位置 |
| Y 坐标 | ✓ | 地图 Y 位置 |
| 角度 | ✓ | 朝向角度 |
| 停车精度 | | 允许偏差（默认 0.1m） |
| 最大等待时间 | | 超时处理 |

### 3.4 编辑站点

可以修改的信息：
- 站点名称
- 站点类型
- 站点坐标（微调）
- 站点属性参数

不可随意修改：
- 站点编号
- 已关联的任务配置

## 4. 路径配置

### 4.1 路径类型

| 类型 | 说明 | 配置方式 |
|------|------|----------|
| 直线路径 | 两点间直线 | 自动生成 |
| 弧线路径 | 圆弧转弯 | 圆心+半径 |
| 复杂路径 | 多段组合 | 手动绘制 |

### 4.2 创建路径

**方式一：自动生成**

```
适用于：两站点间无障碍
步骤：
1. 选择起点站点
2. 选择终点站点
3. 点击"生成路径"
4. 系统自动规划最短路径
5. 检查路径是否穿过障碍物
6. 如有问题，切换手动调整
```

**方式二：手动绘制**

```
适用于：复杂环境或多条可选路径
步骤：
1. 选择起点
2. 依次点击路径经过的点
3. 每个点称为"路径点"
4. 双击终点完成
5. 调整路径点位置
6. 设置转弯半径和速度
```

### 4.3 路径属性

| 属性 | 说明 |
|------|------|
| 路径长度 | 自动计算 |
| 转弯数量 | 自动统计 |
| 最大速度 | 路径限速 |
| 优先级别 | 多路径时选择 |
| 备注 | 特殊说明 |

### 4.4 路径优化

**速度规划原则：**
```
直线路段：可全速行驶
转弯路段：减速至 30-50%
狭窄路段：减速至 30%
十字路口：观察后通行
```

**路径调整建议：**
- 减少不必要的转弯
- 避免 180° 回转
- 转弯半径不宜过小
- 预留避让空间

## 5. 特殊区域配置

### 5.1 限速区

```
创建方法：
1. 在地图上框选区域
2. 选择"设为限速区"
3. 设置限速值（m/s）
4. 设置生效时段（可选）
5. 保存配置

示例：
- 弯道限速：0.5 m/s
- 狭窄通道：0.3 m/s
- 人流密集区：0.3 m/s
```

### 5.2 禁行区

```
创建方法：
1. 在地图上框选区域
2. 选择"设为禁行区"
3. 设置禁行原因
4. 保存配置

用途：
- 安全风险区域
- 设备维护区域
- 临时施工区域
```

### 5.3 优先通行区

```
用于：多 AGV 交汇处
配置：
- 设置通行优先级
- 配置让行规则
- 设置等待位
```

## 6. 地图验证

### 6.1 功能测试

地图配置完成后必须测试：

```
测试项目：
□ AGV 能精确定位到每个站点
□ 路径规划生成正确路径
□ AGV 沿路径行驶无偏离
□ 障碍物检测正常工作
□ 限速区速度限制生效
□ 禁行区规则生效
```

### 6.2 精度验证

| 验证项目 | 标准 | 方法 |
|----------|------|------|
| 定位精度 | ± 0.05m | 多次到站测试 |
| 停车精度 | ± 0.1m | 测量偏差 |
| 路径偏差 | ± 0.1m | 轨迹对比 |
| 障碍检测距离 | ± 0.05m | 标准障碍测试 |

### 6.3 压力测试

```
测试方法：
1. 多台 AGV 同时运行
2. 模拟高峰作业场景
3. 测试路径冲突处理
4. 验证系统稳定性
5. 测试极端情况处理
```

## 8. 常见问题

**Q: 建图时地图变形怎么办？**
A: 降低建图速度，检查环境一致性，删除问题区域重新扫描

**Q: 站点定位精度差怎么办？**
A: 校准激光传感器，检查反射板位置，重新进行定位校准

**Q: 路径规划失败怎么处理？**
A: 检查是否有未连通的站点，修复拓扑连接，添加中间路径点
