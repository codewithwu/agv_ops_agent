# AGV 传感器校准指南

## 概述

本文档介绍 AGV 各传感器的校准方法，确保传感器精度满足运行要求。

## 1. 传感器系统概览

### 1.1 传感器清单

| 传感器 | 型号 | 功能 | 校准周期 |
|--------|------|------|----------|
| 激光雷达 | SICK TIM571 | 环境感知/定位 | 6 个月 |

| 编码器 | 增量式 1024PPR | 里程测量 | 6 个月 |

## 2. 激光雷达校准

### 2.1 校准前检查

```
□ 镜头清洁，无灰尘污渍
□ 安装支架稳固，无松动
□ 线缆连接正常
□ 供电电压正常（24V ± 5%）
□ 无异常噪音或振动
```

### 2.2 角度校准

**目的：** 确保激光扫描角度与 AGV 行驶方向一致

**步骤：**
```
1. 将 AGV 置于已知位置
2. 在 AGV 正前方 3m 处放置标靶
3. 记录标靶在扫描数据中的角度值
4. 计算实际角度与理论角度的偏差
5. 在配置文件中调整角度偏移
6. 重复测试直到偏差 < 0.5°
```

**角度计算公式：**
```
偏差角 = 实际测量角 - 理论角度
校准偏移 = -偏差角
```

### 2.3 距离校准

**目的：** 修正距离测量误差

**步骤：**
```
1. 在 AGV 正前方 1m、2m、3m、5m 处放置标靶
2. 记录各距离的测量值
3. 计算测量值与实际值的偏差
4. 线性拟合校准曲线
5. 更新校准参数
```

**允许误差范围：**

| 距离 | 最大误差 |
|------|----------|
| 1m | ± 10mm |
| 2m | ± 15mm |
| 5m | ± 30mm |
| 10m | ± 50mm |

## 3. 陀螺仪校准

### 3.1 零偏校准

**目的：** 消除陀螺仪零偏输出

**条件：**
- AGV 静止放置
- 水平地面
- 无振动干扰
- 温度稳定（室内温度）

**步骤：**
```
1. 开启 AGV 系统
2. 等待陀螺仪预热（5 分钟）
3. 进入陀螺仪校准模式
4. 保持 AGV 完全静止
5. 等待校准完成（约 30 秒）
6. 记录零偏值
7. 保存校准参数
```

### 3.2 标度因数校准

**目的：** 修正角度输出比例系数

**步骤：**
```
1. 将 AGV 置于转台上
2. 设置转台旋转 360°
3. 记录陀螺仪输出角度
4. 计算误差比例
5. 调整标度因数
6. 重复测试验证
```

**标度因数计算：**
```
标度因数 = 实际旋转角度 / 陀螺仪输出角度
正常范围：0.9995 - 1.0005
```

### 3.3 温度补偿校准

| 温度点 | 校准方法 |
|--------|----------|
| 0°C | 低温箱测试 |
| 25°C | 常温参考点 |
| 40°C | 高温箱测试 |

建立温度-零偏曲线，用于实时补偿。

## 4. 编码器校准

### 4.1 里程计校准

**目的：** 确保行驶距离测量准确

**步骤：**
```
1. 在地面标记起始点
2. 设置 AGV 沿直线行驶 10m
3. 记录编码器输出脉冲数
4. 计算实际距离与测量距离的偏差
5. 调整比例系数
```

**计算公式：**
```
比例系数 = 实际距离 / 测量距离
校准后距离 = 测量距离 × 比例系数
```

### 4.2 左/右轮同步校准

**目的：** 修正双轮差异导致的曲线行驶

**步骤：**
```
1. 直线行驶 5m，记录左右编码器脉冲数
2. 计算左右脉冲数差异
3. 如果差异 > 1%，调整轮径参数
4. 重复测试直到直线行驶无偏斜
```

## 5. 超声波传感器校准

### 5.1 距离校准

**步骤：**
```
1. 在传感器正前方放置平面障碍物
2. 距离分别为 0.2m、0.4m、0.6m、1.0m
3. 记录各距离的测量值
4. 计算偏差并校准
```

**允许误差：± 10mm**

### 5.2 检测角度校准

**步骤：**
```
1. 以传感器为中心，在 ±15°、±30° 放置障碍物
2. 检查是否能可靠检测
3. 记录盲区角度范围
4. 在导航算法中设置盲区参数
```

## 6. 碰撞传感器校准

### 6.1 触发阈值校准

**目的：** 确保碰撞检测灵敏度合适

**步骤：**
```
1. 使用标准力计施加不同大小的力
2. 记录触发时的力值
3. 调整弹簧预紧力或触发位置
4. 确保既不灵敏过度又不漏触发
```

**触发力标准：**
| 位置 | 触发力 |
|------|--------|
| 前保险杠 | 15-25N |
| 后保险杠 | 15-25N |
| 侧边防撞条 | 20-30N |

### 6.2 响应时间测试

**标准：** 触发到输出信号 < 50ms

## 7. 地面传感器校准

### 7.1 反射式传感器校准

**目的：** 确保地面标记检测可靠

**步骤：**
```
1. 在黑色背景（无标记）上测试
2. 记录输出信号值（应为低）
3. 在白色标记上测试
4. 记录输出信号值（应为高）
5. 调整阈值到中间值
6. 测试确认能可靠区分
```

**阈值计算：**
```
阈值 = (黑信号 + 白信号) / 2
```

### 7.2 地面标记定位精度

**测试方法：**
```
1. 沿标记线行驶
2. 记录每次检测到的位置
3. 计算与标称位置的偏差
4. 偏差应 < 20mm
```
