# AGV 调度系统使用指南

## 概述

本文档介绍 AGV 调度系统的操作方法，包括任务下发、状态监控、路径配置等功能。

## 1. 系统登录

### 1.1 登录方式

```
访问地址：http://{服务器IP}:8080/ops
默认端口：8080
```

### 1.2 登录凭证

| 角色 | 用户名 | 密码 | 权限 |
|------|--------|------|------|
| 管理员 | admin | （由系统管理员分配） | 全部功能 |
| 操作员 | operator | （由系统管理员分配） | 任务操作 |
| 监控员 | viewer | （由系统管理员分配） | 仅监控 |

### 1.3 登录流程

1. 输入用户名和密码
2. 选择语言（中文/English）
3. 点击"登录"按钮
4. 首次登录需修改默认密码

## 3. 地图管理

### 3.1 地图查看

在地图界面可以：
- 查看所有 AGV 实时位置
- 查看站点位置和名称
- 查看路径规划
- 查看障碍物分布

### 3.2 站点管理

**添加站点：**
```
步骤：
1. 点击地图空白处
2. 选择"添加站点"
3. 输入站点编号（唯一）
4. 输入站点名称
5. 设置站点类型（起点/终点/中转/充电）
6. 点击"保存"
```

**编辑站点：**
```
步骤：
1. 点击已有站点
2. 选择"编辑站点"
3. 修改相关信息
4. 点击"保存"
```

**删除站点：**
```
注意：删除前需确认无任务关联
步骤：
1. 点击目标站点
2. 选择"删除站点"
3. 确认删除操作
```

### 3.3 路径配置

**设置路径点：**
```
步骤：
1. 选择起点站点
2. 选择"创建路径"
3. 依次点击路径经过的点
4. 双击终点站点完成
5. 系统自动计算距离和时间
```

## 4. 任务管理

### 4.1 下发任务

**方式一：快速下单**
```
1. 选择目标 AGV（或选择自动分配）
2. 选择起点站点
3. 选择终点站点
4. 设置优先级（普通/加急）
5. 点击"下发任务"
```

**方式二：批量任务**
```
1. 点击"批量任务"
2. 上传任务 CSV 文件
3. 格式：AGV编号,起点,终点,优先级
4. 点击"确认导入"
5. 检查预览后执行
```

**任务参数说明：**

| 参数 | 可选值 | 说明 |
|------|--------|------|
| AGV | 自动/指定编号 | 自动分配空闲 AGV |
| 起点 | 地图上的站点 | 默认当前位置 |
| 终点 | 地图上的站点 | 必填 |
| 优先级 | 普通/加急/紧急 | 影响调度排序 |
| 执行时间 | 立即/定时 | 定时任务设置时间 |

### 4.2 任务监控

**任务状态流转：**
```
待执行 → 执行中 → 已完成
            ↓
         已取消
            ↓
         执行失败
```

**任务列表筛选：**
- 按状态筛选（全部/待执行/进行中/已完成/已取消）
- 按 AGV 编号筛选
- 按时间范围筛选
- 按优先级筛选

### 4.3 任务操作

| 操作 | 适用状态 | 说明 |
|------|----------|------|
| 取消任务 | 待执行/执行中 | 立即终止任务 |
| 暂停任务 | 执行中 | 暂停 AGV 移动 |
| 恢复任务 | 暂停 | 继续执行任务 |
| 改派任务 | 执行中 | 切换到其他 AGV |

## 5. AGV 状态监控

### 5.1 状态面板

点击 AGV 图标查看详细信息：

```
┌─────────────────────────────┐
│  AGV-01 状态详情            │
├─────────────────────────────┤
│  状态：运行中                │
│  任务：任务#20260417001     │
│  当前位置：站点A → 站点B    │
│  进度：65%                  │
│  电池：72%                  │
│  速度：1.2 m/s              │
│  温度：38°C                 │
├─────────────────────────────┤
│  [远程控制] [查看轨迹] [详情]│
└─────────────────────────────┘
```

### 5.2 远程控制

对在线 AGV 可执行远程操作：

| 操作 | 说明 | 权限要求 |
|------|------|----------|
| 启动 | 从停靠点启动 | 操作员 |
| 停止 | 立即停止移动 | 操作员 |
| 暂停 | 暂停当前任务 | 操作员 |
| 继续 | 恢复暂停的任务 | 操作员 |
| 回充 | 强制前往充电站 | 操作员 |
| 复位 | 故障后复位 | 管理员 |

### 5.3 实时轨迹

查看 AGV 历史轨迹：
```
1. 选择 AGV 和时间范围
2. 点击"查看轨迹"
3. 系统显示运动轨迹线
4. 可播放历史动画
5. 支持导出轨迹数据
```

## 6. 告警管理

### 6.1 告警级别

| 级别 | 标识 | 声音 | 处理时限 |
|------|------|------|----------|
| 严重 | 🔴 红色 | 持续 | 立即处理 |
| 警告 | 🟡 黄色 | 短促 | 30 分钟内 |
| 提示 | 🔵 蓝色 | 无 | 计划处理 |

### 6.2 常见告警

| 告警内容 | 级别 | 可能原因 | 处理建议 |
|----------|------|----------|----------|
| AGV 离线 | 严重 | 网络中断/电源故障 | 立即检查 |
| 电池电量低 | 警告 | 需要充电 | 安排回充 |
| 路径阻塞 | 警告 | 障碍物阻挡 | 清除障碍 |
| 定位丢失 | 严重 | 传感器异常 | 重新定位 |
| 碰撞告警 | 严重 | 障碍检测触发 | 安全确认 |

### 6.3 告警处理

```
告警处理流程：
1. 收到告警后点击查看详情
2. 确认告警原因
3. 采取相应措施
4. 处理完成后点击"消警"
5. 记录处理过程
```

## 8. 系统设置

### 8.1 用户管理

| 功能 | 说明 |
|------|------|
| 添加用户 | 输入用户名、密码、角色 |
| 编辑用户 | 修改用户信息 |
| 删除用户 | 删除前需确认无进行中任务 |
| 重置密码 | 管理员可强制重置 |

### 8.2 参数配置

| 参数 | 默认值 | 可调范围 | 说明 |
|------|--------|----------|------|
| 最大同时运行数 | 10 | 1-50 | 系统同时运行 AGV 数量 |
| 默认速度 | 1.0 m/s | 0.3-1.5 | 正常运行速度 |
| 低电量阈值 | 20% | 10-30% | 触发回充的电量 |
| 障碍检测距离 | 2.0 m | 0.5-5.0 | 安全停止距离 |

### 8.3 地图备份

- 建议每周备份一次地图配置
- 重要修改前立即备份
- 备份文件保留最近 10 个版本
