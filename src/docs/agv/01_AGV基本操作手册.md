# AGV 基本操作手册

## 概述

本文档介绍 AGV（自动导引车）的基本操作方法，适用于工厂运维人员日常操作。

## 1. AGV 启动与停止

### 1.1 启动流程

1. **检查环境安全**
   - 确认运行路线无障碍物
   - 检查充电桩是否正常
   - 确认急停按钮未被触发

2. **系统上电**
   - 打开 AGV 主电源开关
   - 等待系统自检完成（约 30 秒）
   - 确认显示屏无故障代码

3. **连接调度系统**
   - AGV 自动连接中央调度服务器
   - 状态指示灯变为绿色常亮
   - 调度系统显示 AGV 在线

### 1.2 正常停止流程

1. 将 AGV 行驶至指定停靠点
2. 在调度系统中提交"待命"状态
3. 关闭 AGV 电源开关
4. 填写设备运行记录表

### 1.3 紧急停止

- **立即按下** 红色急停按钮
- 切断 AGV 主驱动电源
- 记录故障时间和现象
- 通知维护人员检查

## 2. 手动控制模式

### 2.1 进入手动模式条件

- AGV 处于停止状态
- 急停按钮已复位
- 电池电量 > 20%

### 2.2 手动操作步骤

1. 在控制面板选择"手动模式"
2. 使用摇杆或方向键控制行驶
3. 最高速度限制为 0.3m/s
4. 手动模式仅用于调试或故障处理

### 2.3 注意事项

- 手动模式运行时 AGV 不具备避障功能
- 必须有人员全程监控
- 严禁在自动运行区域使用手动模式

## 3. 任务下发与监控

### 3.1 任务下发

通过调度系统下发任务：
- 选择 AGV 编号
- 输入目的站点编号
- 设置优先级（普通/紧急）
- 确认任务参数

### 3.2 状态监控

监控以下关键指标：
| 指标 | 正常范围 | 异常处理 |
|------|----------|----------|
| 电池电量 | 20% - 100% | 低于 20% 自动回充 |
| 运行速度 | 0 - 1.5m/s | 超速需检查电机 |
| 通讯信号 | 满格 | 信号弱检查天线 |
| 障碍检测 | 无告警 | 立即停车检查 |

## 4. 日常检查项目

### 4.2 运行前检查

- [ ] 外观检查：无明显损坏
- [ ] 轮胎检查：磨损正常，气压充足
- [ ] 传感器检查：清洁无遮挡
- [ ] 电池检查：电量 > 50%

### 4.2 运行后检查

- [ ] 记录当日运行里程
- [ ] 检查路径偏移情况
- [ ] 清理驱动轮缠杂物
- [ ] 确认停靠位置准确

## 5. 常见操作问答

**Q: AGV 无法连接调度系统怎么办？**
A: 检查网络连接、确认 AGV 的 IP 配置、重启 AGV 控制系统

**Q: 任务下发后 AGV 不执行？**
A: 检查 AGV 状态是否为"在线"、确认地图数据完整、验证路径可行性

**Q: 如何强制取消当前任务？**
A: 在调度系统中选择"取消任务"，或直接按下 AGV 急停按钮
